# Secure Notes App

Client-side secure notes application with lock and unlock logic.

## Features
- Passcode-based access control
- Note encryption (Base64 demo)
- Persistent secure storage
- Lock / unlock state handling

## Usage
Open `index.html`, enter the passcode, and manage secure notes.

## Note
This is a demo implementation for learning purposes.

## Purpose
Demonstrates security concepts, state control, and protected data access.
